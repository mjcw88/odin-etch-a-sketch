# The Odin Project (Foundations) - Etch A Sketch

Etch A Sketch project for the Odin Project, featuring extensive DOM manipulation.

Hovering the mouse over the grid will colour it in depending on which button is pressed, by default this will be the "colour" button.

##### Resize

Allows the user to resize the grid anyway from 1x1 to 100x100.

##### Colour

Colour mode will colour in the squares based on the colour selected from the colour picker to the left of this button. Clicking the colour picker allows the user to change colour.

##### Random

Random mode will generate a random colour every time the user hovers over a square.

##### Darken

Darken mode will darken the square's opacity by 10% every time the user hovers over the square.

##### Eraser

Eraser mode will erase the colour from the square.

##### Grid

The grid button toggles the grid lines visibility.

##### Clear

The clear button clears the entire grid.
